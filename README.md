## ğŸ“– Documentation

### ğŸµ Track Management

#### JSON Track Format

```javascript
{
  "title": "Song Title",
  "singer": "Artist Name",
  "writer": "Lyricist Name",
  "theme": "cyberpunk",           // Template name
  "color_mode": "rainbow_char",   // Color mode
  "solid_color": "cyan",          // For solid mode only
  "lyrics": [
    "Line 1 of lyrics",
    "Line 2 of lyrics",
    "Line 3 of lyrics"
  ],
  "intervals": [3.0, 3.2, 3.5],   // Seconds per line
  "effects": true                  // Enable effects
}
```

#### Available Theme Values

```javascript
"theme": "cyberpunk" | "fire" | "ocean" | "forest" | "sunset" 
       | "neon" | "vaporwave" | "matrix" | "synthwave" | "minimal"
```

#### Available Color Modes

```javascript
"color_mode": "rainbow_char" | "rainbow_word" | "gradient" 
            | "alternate" | "random" | "wave" | "solid"
```

#### Solid Color Options

```javascript
"solid_color": "cyan" | "magenta" | "yellow" | "green" | "red" 
             | "blue" | "purple" | "orange" | "pink" | "white"
```
